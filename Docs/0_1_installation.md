# Installation {#installation}

## Requirements
* Unity 6.0 or later

### From Unity Asset Store
The package is still in alpha state and will be released on the asset store as a free asset once we get to version 1.0.0

### From GitHub
Download the latest release from our GitHub repository: [https://github.com/GarageWareGames/GWG-USOUIElements/](https://github.com/GarageWareGames/GWG-USOUIElements/) and extract the contents into your Unity project's Assets folder.
